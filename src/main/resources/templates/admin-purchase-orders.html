Automatic PO-GRN Integration

How the New System Works:

  • DRAFT → PENDING_APPROVAL → APPROVED: Manual status progression with approval workflow
  • APPROVED → ORDERED: Automatic when GRN is created (items received)
  • ORDERED → PARTIALLY_RECEIVED: Automatic when some items are received via GRN
  • PARTIALLY_RECEIVED → FULLY_RECEIVED: Automatic when all items are received
  • Stock Management: Stock is reserved when PO is approved and updated when GRNs are inspected

Note: PO status automatically updates based on GRN creation and receipt quantities. Use the "Receipt" button to view detailed receipt summaries.

Delete option is only available for Purchase Orders in DRAFT or CANCELLED status
